## Reparar automaticamente

```
npx eslint . --fix
```

## Reparar errores de eslint automaticamente

1. tener instalado la extensión eslint
2. Configurar en settings lo siguiente

![](2022-01-26-11-00-34.png)

```
"editor.codeActionsOnSave": {
    "source.fixAll": true
  },
```
